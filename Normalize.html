All output must only contain a–z, - (hyphen), and ' (apostrophe). Words cannot start or end with a hyphen.


Title abbreviations — absorb a following period and space:

Punctuation quick reference:

/ \ → hyphen (merges with adjacent -)  ·  ~ _ → hyphen (no merge, ignored at edges)  ·  . → hyphen (any count = 1)  ·  , → hyphen (ignored at edges)  ·  : → hyphen (:: = colon-colon, ::: = colon-colon-colon)  ·  ; → hyphen (;; = semicolon / semi-colon variants)  ·  + → plus (each + = own segment)  ·  () [] {} → paired = hyphen, unpaired = parenthesis / bracket / curly-bracket  ·  ! → exclamation-mark / exclamation-point / omit  ·  ? → question-mark / omit
